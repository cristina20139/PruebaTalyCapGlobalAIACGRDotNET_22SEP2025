Pregunta 7 – Opción Múltiple

¿Para qué es usada la herencia en un lenguaje de P.O.O como Java?

  • a) Permite la replicación de comportamientos y atributos de una clase padre sobre una clase hija
  • b) Permite la aplicación de polimorfismo en la definición del programa
  • c) Permite el uso de comportamientos y atributos en la clase hija
  • d) Todas las respuestas son correctas

Respuesta Correcta:

d) Todas las respuestas son correctas

Justificación:

En Java, la herencia permite que una clase hija (subclase) reutilice atributos y métodos de una clase padre (superclase), evitando duplicación de código (opción a). También permite que la clase hija pueda sobrescribir o extender el comportamiento de la clase padre, facilitando el polimorfismo (opción b). Finalmente, la herencia habilita el uso directo de los métodos y atributos de la clase padre en la clase hija (opción c). Por lo tanto, todas las respuestas son correctas.

Nota: La herencia es uno de los pilares de la programación orientada a objetos y se combina frecuentemente con polimorfismo para crear código más flexible y reutilizable.

Ejemplo en Java:

class Vehiculo {
    protected String marca;

    public Vehiculo(String marca) {
        this.marca = marca;
    }

    public void mostrarMarca() {
        System.out.println("Marca: " + marca);
    }
}

// Clase hija que hereda de Vehiculo
class Auto extends Vehiculo {
    private int puertas;

    public Auto(String marca, int puertas) {
        super(marca); // Uso del constructor de la clase padre
        this.puertas = puertas;
    }

    // Sobrescritura de método (polimorfismo)
    @Override
    public void mostrarMarca() {
        System.out.println("Auto de marca: " + marca + " con " + puertas + " puertas");
    }
}

public class Main {
    public static void main(String[] args) {
        Vehiculo v = new Vehiculo("Genérico");
        v.mostrarMarca();

        Auto miAuto = new Auto("Toyota", 4);
        miAuto.mostrarMarca(); // Método sobrescrito
    }
}
