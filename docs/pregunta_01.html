Pregunta 1 – Opción Múltiple

Es un tipo de dato que permite almacenar enteros con una precisión de 264 – 1:

  • a) long
  • b) int
  • c) double
  • d) Todas las respuestas son correctas

Respuesta Correcta:

a) long

Justificación:

En Java, el tipo long es un entero de 64 bits con signo que puede almacenar valores desde -263 hasta 263 - 1. El tipo int tiene solo 32 bits, por lo que no alcanza la precisión de 264 - 1. El tipo double es un número de punto flotante de 64 bits, que puede representar valores muy grandes pero no con precisión entera exacta para todos los valores dentro de ese rango.

Nota: Siempre que necesites almacenar enteros extremadamente grandes, utiliza long en lugar de int o double.

Ejemplo en Java:

public class Main {
    public static void main(String[] args) {
        long numeroGrande = 9223372036854775807L; // Valor máximo de long
        System.out.println("Número grande: " + numeroGrande);

        int numeroInt = 2147483647; // Valor máximo de int
        System.out.println("Número int máximo: " + numeroInt);

        double numeroDouble = 9.223372036854776E18; // Representación aproximada de long en double
        System.out.println("Número double aproximado: " + numeroDouble);
    }
}
