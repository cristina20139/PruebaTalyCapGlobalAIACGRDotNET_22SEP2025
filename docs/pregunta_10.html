Pregunta 10 – Opción Múltiple

La función de Java .split(-argumento-), aplicada sobre cadenas, separa dicha cadena basada en el delimitador definido como argumento y retorna un arreglo de tamaño n. Con base a ello, si a la cadena “Hola;bienvenido;a;la;aplicación;” se le aplica la función .split() usando como argumento el símbolo "\\;", ¿cuál es el tamaño del arreglo resultante?

  • a) 5
  • b) 7
  • c) 6
  • d) Arroja error de compilación

Respuesta Correcta:

c) 6

Justificación:

El método split() de Java separa la cadena usando el delimitador especificado y retorna un arreglo con todos los elementos entre los delimitadores. En la cadena "Hola;bienvenido;a;la;aplicación;", hay 5 puntos y 6 segmentos separados: "Hola", "bienvenido", "a", "la", "aplicación", y un segmento vacío "" al final debido al último ;. Por ello, el tamaño del arreglo resultante es 6.

Nota: En Java, split acepta expresiones regulares, por eso se usa "\\;" para escapar el carácter punto y coma.

Ejemplo en Java:

public class Main {
    public static void main(String[] args) {
        String texto = "Hola;bienvenido;a;la;aplicación;";
        
        // Separando por el delimitador ;
        String[] partes = texto.split("\\;");

        System.out.println("Número de elementos: " + partes.length);

        for(int i = 0; i < partes.length; i++) {
            System.out.println("Elemento " + i + ": '" + partes[i] + "'");
        }
    }
}
