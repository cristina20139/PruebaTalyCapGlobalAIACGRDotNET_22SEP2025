Pregunta 3 – Opción Múltiple

La declaración de una lista de datos va denotada por la sintaxis:

  • a) List<String> lista = new List<String>();
  • b) List<String> lista = new ArrayList<String>();
  • c) List lista = new ArrayList();
  • d) Todas las respuestas son correctas

Respuesta Correcta:

b) List<String> lista = new ArrayList<String>();

Justificación:

En Java, List es una interfaz y no se puede instanciar directamente, por lo que la opción a) es incorrecta. La forma correcta de declarar e instanciar una lista de Strings es usando una clase que implemente la interfaz List, como ArrayList. La opción c) es válida sintácticamente pero no usa generics, lo que reduce la seguridad de tipo y no es recomendable. Por eso, la opción más correcta y segura es la b).

Nota: Usar generics permite que la lista solo contenga elementos del tipo especificado y evita errores de casting en tiempo de ejecución.

Ejemplo en Java:

import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        // Declaración correcta usando List e ArrayList con generics
        List<String> lista = new ArrayList<>();
        lista.add("Manuela");
        lista.add("Carlos");
        lista.add("Ana");

        for(String nombre : lista){
            System.out.println("Nombre: " + nombre);
        }

        // Declaración sin generics (opción c)
        List listaSinTipo = new ArrayList();
        listaSinTipo.add("Texto");
        listaSinTipo.add(123); // Esto es posible pero menos seguro
        System.out.println("Lista sin generics: " + listaSinTipo);
    }
}
