Pregunta 6 – Opción Múltiple

Es una instrucción de Java aplicada sobre cadenas que recibe un carácter como argumento y retorna la posición del carácter si existe en la cadena; retorna -1 en otros casos:

  • a) indexOf(-argumento-)
  • b) trim(-argumento-)
  • c) contains(-argumento-)
  • d) Todas las respuestas son correctas

Respuesta Correcta:

a) indexOf(-argumento-)

Justificación:

En Java, el método indexOf() se aplica sobre un String y retorna la posición de la primera aparición de un carácter o subcadena dentro de la cadena. Si el carácter o subcadena no se encuentra, retorna -1. La opción b) trim() elimina espacios en blanco al inicio y al final de la cadena, y c) contains() retorna un valor booleano indicando si la subcadena existe, pero no su posición. Por lo tanto, la opción correcta es a).

Nota: indexOf() es muy útil para localizar caracteres o subcadenas dentro de un String.

Ejemplo en Java:

public class Main {
    public static void main(String[] args) {
        String texto = "Hola Mundo";

        // indexOf devuelve la posición del carácter
        int pos = texto.indexOf('M'); // retorna 5
        System.out.println("Posición de 'M': " + pos);

        // indexOf de un carácter que no existe
        int posNoExiste = texto.indexOf('z'); // retorna -1
        System.out.println("Posición de 'z': " + posNoExiste);

        // trim elimina espacios en los extremos
        String conEspacios = "   Hola   ";
        System.out.println("Trim: '" + conEspacios.trim() + "'");

        // contains retorna boolean
        boolean tieneMundo = texto.contains("Mundo");
        System.out.println("Contiene 'Mundo': " + tieneMundo);
    }
}
