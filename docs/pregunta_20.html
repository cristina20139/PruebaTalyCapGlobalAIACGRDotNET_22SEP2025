Pregunta 20 – Factorial de un número

A continuación se presentan dos problemas de aplicación. Seleccione uno y aporte una solución precisa.

He elegido la opción b):

  • a) Ordenar un arreglo de datos {2, 4, 5, 1, 6, 3, 9, 7, 8, 0} en orden ascendente.
  • b) Calcular el factorial de un número positivo n.

Nota:

No se permite la implementación de métodos propios de Java para el ordenamiento (ej.: Arrays.sort()). Se permite el uso de recursos tecnológicos solamente para consulta de conceptos.

Solución – Cálculo de factorial:

Se puede implementar de manera iterativa o recursiva. A continuación se presenta un ejemplo iterativo:

import java.util.Scanner;

public class Factorial {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Ingrese un número positivo: ");
        int n = sc.nextInt();
        
        if (n < 0) {
            System.out.println("Número inválido. Debe ser positivo.");
        } else {
            long factorial = 1;
            for (int i = 1; i <= n; i++) {
                factorial *= i;
            }
            System.out.println("El factorial de " + n + " es: " + factorial);
        }
        
        sc.close();
    }
}

Explicación:

  • Se solicita al usuario un número positivo n.
  • Se valida que n sea positivo.
  • Se calcula el factorial mediante un bucle for multiplicando desde 1 hasta n.
  • Se imprime el resultado.
Nota: Este algoritmo evita el uso de métodos complejos de librerías y cumple la condición de ser un programa sencillo y preciso en Java.
