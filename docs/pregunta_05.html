Pregunta 5 – Opción Múltiple

La manera de definir un método constructor para una clase “Vehiculo” en el lenguaje Java es:

  • a) public Vehiculo(){}
  • b) public void init(){ }
  • c) public Vehiculo(-argumentos-){}
  • d) Todas las respuestas son correctas

Respuesta Correcta:

a) public Vehiculo(){} y c) public Vehiculo(-argumentos-){}

Justificación:

En Java, un constructor es un método especial cuyo nombre debe coincidir exactamente con el nombre de la clase y no tiene tipo de retorno, ni siquiera void. La opción b) es incorrecta, ya que declarar public void init() es un método normal, no un constructor. Por tanto, los constructores válidos son aquellos que coinciden con el nombre de la clase, con o sin argumentos, como en las opciones a) y c).

Nota: Un constructor sin argumentos se llama constructor por defecto, mientras que uno con parámetros permite inicializar objetos con valores específicos.

Ejemplo en Java:

public class Vehiculo {
    private String marca;
    private int año;

    // Constructor sin argumentos
    public Vehiculo() {
        this.marca = "Desconocida";
        this.año = 0;
    }

    // Constructor con argumentos
    public Vehiculo(String marca, int año) {
        this.marca = marca;
        this.año = año;
    }

    public void mostrarInfo() {
        System.out.println("Marca: " + marca + ", Año: " + año);
    }

    public static void main(String[] args) {
        // Usando constructor sin argumentos
        Vehiculo v1 = new Vehiculo();
        v1.mostrarInfo();

        // Usando constructor con argumentos
        Vehiculo v2 = new Vehiculo("Toyota", 2025);
        v2.mostrarInfo();
    }
}
