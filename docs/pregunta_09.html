Pregunta 9 – Opción Múltiple

La declaración de importación de clases ajenas en Java va denotada por la línea “import – ruta del paquete - - nombre de la clase –“. Con base a ello, si se desea utilizar todas las clases del paquete java.util, ¿la sintaxis correcta para importar el paquete es?

  • a) import java.utils.*;
  • b) mport java.util.*;
  • c) import java.util.*;
  • d) Ninguna de las respuestas es correcta

Respuesta Correcta:

c) import java.util.*;

Justificación:

En Java, la palabra clave import seguida de la ruta del paquete y un asterisco * permite importar todas las clases contenidas en ese paquete. La opción a) es incorrecta porque el paquete correcto es java.util, no java.utils. La opción b) es incorrecta debido a un error de sintaxis: falta la letra i al inicio de import. Por lo tanto, la opción correcta es c).

Nota: Importar un paquete con * no importa subpaquetes; solo las clases que se encuentran directamente dentro del paquete especificado.

Ejemplo en Java:

import java.util.*;

public class Main {
    public static void main(String[] args) {
        // Usando ArrayList del paquete java.util
        List lista = new ArrayList<>();
        lista.add("Manuela");
        lista.add("Carlos");
        lista.add("Ana");

        for(String nombre : lista) {
            System.out.println("Nombre: " + nombre);
        }

        // Usando HashMap del mismo paquete
        Map edades = new HashMap<>();
        edades.put("Manuela", 30);
        edades.put("Carlos", 35);

        System.out.println("Edades: " + edades);
    }
}
