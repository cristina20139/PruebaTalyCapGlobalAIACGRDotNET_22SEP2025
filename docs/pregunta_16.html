Pregunta 16 – Try-Catch vs Throws

¿Qué diferencia hay entre un bloque try-catch y un bloque throws? Dar un ejemplo.

Respuesta:

- try-catch: Se usa para capturar y manejar excepciones dentro del bloque de código donde pueden ocurrir. - throws: Se usa en la declaración de un método para indicar que puede lanzar una o varias excepciones, delegando su manejo al método que lo llame.

Ejemplo en Java:

1. Ejemplo con try-catch

public class EjemploTryCatch {
    public static void main(String[] args) {
        try {
            int resultado = 10 / 0; // Puede lanzar ArithmeticException
        } catch (ArithmeticException e) {
            System.out.println("Error: División entre cero detectada.");
        }
    }
}

2. Ejemplo con throws

public class EjemploThrows {
    public static void main(String[] args) {
        try {
            leerArchivo("archivo.txt");
        } catch (Exception e) {
            System.out.println("Error al leer el archivo: " + e.getMessage());
        }
    }

    public static void leerArchivo(String nombreArchivo) throws Exception {
        // Simulando excepción
        throw new Exception("Archivo no encontrado");
    }
}
Nota: En try-catch el manejo ocurre localmente, mientras que throws delega la responsabilidad al llamador del método.
