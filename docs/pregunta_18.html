Pregunta 18 – Variable Primitiva vs Wrapper

¿Qué diferencia hay entre una variable de datos primitiva y una variable de datos envolvente (wrapper) en Java?

Respuesta:

- Variable primitiva: Es un tipo de dato básico incorporado en Java (int, boolean, double, char, etc.). Ocupa menos memoria, tiene acceso rápido y no es un objeto. - Variable envolvente (wrapper): Es una clase que encapsula un valor primitivo en un objeto (Integer, Boolean, Double, Character, etc.). Permite usar métodos, trabajar con colecciones y soporta null.

Ejemplo en Java:

1. Variable primitiva

int numero = 10;
double valor = 5.5;
boolean esActivo = true;
System.out.println("Número: " + numero + ", Valor: " + valor + ", Activo: " + esActivo);

2. Variable wrapper

Integer numeroObj = Integer.valueOf(10);
Double valorObj = Double.valueOf(5.5);
Boolean esActivoObj = Boolean.TRUE;

// Permite usar métodos de la clase
System.out.println("Número doble: " + numeroObj.doubleValue());
System.out.println("Valor como String: " + valorObj.toString());
System.out.println("Activo: " + esActivoObj);
Nota: Las variables wrapper son útiles cuando se trabaja con colecciones genéricas, ya que estas solo pueden contener objetos, no tipos primitivos.
