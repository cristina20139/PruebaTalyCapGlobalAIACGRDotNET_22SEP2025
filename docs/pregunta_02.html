Pregunta 2 – Opción Múltiple

No es una estructura de ciclos del lenguaje Java:

  • a) for(int i = 0; i < limite; i++){ }
  • b) while(int i < limite){ }
  • c) for(String s : listaDatos){}
  • d) Todas son estructuras de iteración

Respuesta Correcta:

b) while(int i < limite){ }

Justificación:

En Java, la estructura while no permite declarar directamente una variable dentro de los paréntesis de condición. La sintaxis correcta para while es primero declarar la variable fuera del ciclo, y luego usar solo la condición dentro del paréntesis. Por ejemplo:

Nota: Las estructuras válidas de iteración en Java incluyen for, while y for-each (enhanced for). Sin embargo, la opción b) tiene una sintaxis inválida.

Ejemplo en Java:

public class Main {
    public static void main(String[] args) {
        int limite = 5;

        // Ciclo for clásico
        for(int i = 0; i < limite; i++){
            System.out.println("For clásico i = " + i);
        }

        // Ciclo while correcto
        int j = 0; // variable declarada fuera del while
        while(j < limite){
            System.out.println("While j = " + j);
            j++;
        }

        // Ciclo for-each
        String[] listaDatos = {"A", "B", "C"};
        for(String s : listaDatos){
            System.out.println("For-each valor: " + s);
        }
    }
}
