Pregunta 12 – Opción Múltiple

En Java, es la instrucción usada para obtener el tamaño de una lista de datos:

  • a) .length();
  • b) .size();
  • c) .iterator();
  • d) Ninguna de las respuestas es correcta

Respuesta Correcta:

b) .size();

Justificación:

En Java, las listas que implementan la interfaz List (como ArrayList o LinkedList) utilizan el método size() para obtener la cantidad de elementos almacenados. La opción a) .length() es propia de arreglos (array.length no lleva paréntesis). La opción c) .iterator() devuelve un iterador para recorrer la lista, pero no su tamaño. Por lo tanto, la forma correcta de obtener el tamaño de una lista es .size().

Nota: size() es un método de la interfaz Collection, y está disponible para todas las estructuras de datos que la implementan.

Ejemplo en Java:

import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List nombres = new ArrayList<>();
        nombres.add("Manuela");
        nombres.add("Carlos");
        nombres.add("Ana");

        // Obtener tamaño de la lista
        int tamaño = nombres.size();
        System.out.println("Tamaño de la lista: " + tamaño); // Salida: 3

        // Recorrer lista
        for(String nombre : nombres) {
            System.out.println("Nombre: " + nombre);
        }
    }
}
