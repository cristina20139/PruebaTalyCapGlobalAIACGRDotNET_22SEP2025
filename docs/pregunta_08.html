Pregunta 8 – Opción Múltiple

Si usted desea imprimir una línea nueva por consola en Java, ¿Qué comando ejecutaría?

  • a) System.out.printf("Nueva línea");
  • b) System.out.print("Nueva línea");
  • c) System.out.println("Nueva línea");
  • d) Todas las anteriores

Respuesta Correcta:

c) System.out.println("Nueva línea");

Justificación:

En Java, System.out.println() imprime el texto en consola y automáticamente agrega un salto de línea al final. System.out.print() imprime el texto pero no agrega salto de línea, y System.out.printf() es usado para formato de salida, pero tampoco agrega salto de línea automáticamente a menos que se incluya \n en el formato. Por lo tanto, la opción correcta para imprimir y pasar a la línea siguiente es System.out.println().

Nota: Para printf se podría usar System.out.printf("Nueva línea%n"); para obtener el mismo efecto de salto de línea.

Ejemplo en Java:

public class Main {
    public static void main(String[] args) {
        // Imprime sin salto de línea
        System.out.print("Primera línea ");
        System.out.print("continuación de la misma línea\n");

        // Imprime con salto de línea automáticamente
        System.out.println("Segunda línea con println");

        // Usando printf con salto de línea
        System.out.printf("Tercera línea con printf%n");
    }
}
