📄 GlobalExceptionHandler

Paquete: com.clinicos.backend.api.rest.infrastructure.controller

🔹 Descripción: Manejador global de excepciones para la aplicación. Centraliza el manejo de errores de todos los controladores REST siguiendo el patrón @RestControllerAdvice en Spring Boot. Permite retornar códigos HTTP consistentes y mensajes claros.

✨ Principios aplicados

  • 📌 SRP (Single Responsibility): Esta clase solo maneja excepciones.
  • 📌 OCP (Open/Closed): Se pueden agregar nuevos handlers sin modificar los existentes.
  • 📌 DIP (Dependency Inversion): Los controladores dependen de este manejo global en lugar de lógica repetitiva.

🛠️ Métodos documentados

🔹 handleBadRequest

Maneja errores de validación o parámetros inválidos.

  • 🔸 Devuelve HTTP 400 (Bad Request)
  • 🔸 Registra log de advertencia con el detalle
  • 🔸 Parámetro: IllegalArgumentException
  • 🔸 Retorno: ResponseEntity<String>

🔹 handleNotFound

Maneja casos donde el recurso no existe.

  • 🔸 Devuelve HTTP 404 (Not Found)
  • 🔸 Registra log informativo
  • 🔸 Parámetro: NoSuchElementException
  • 🔸 Retorno: ResponseEntity<String>

🔹 handleRuntime

Maneja cualquier RuntimeException no capturada.

  • 🔸 Devuelve HTTP 500 (Internal Server Error)
  • 🔸 Centraliza errores inesperados
  • 🔸 Registra log de error completo con stacktrace
  • 🔸 Parámetro: RuntimeException
  • 🔸 Retorno: ResponseEntity<String>

💻 Código Fuente


@RestControllerAdvice
public class GlobalExceptionHandler {

    private static final Logger logger = LoggerFactory.getLogger(GlobalExceptionHandler.class);

    @ExceptionHandler(IllegalArgumentException.class)
    public ResponseEntity<String> handleBadRequest(IllegalArgumentException ex) {
        logger.warn("❌ Error de validación: {}", ex.getMessage(), ex);
        return ResponseEntity.badRequest()
                .body("❌ Error de validación: " + ex.getMessage());
    }

    @ExceptionHandler(NoSuchElementException.class)
    public ResponseEntity<String> handleNotFound(NoSuchElementException ex) {
        logger.info("❌ Recurso no encontrado", ex);
        return ResponseEntity.status(HttpStatus.NOT_FOUND)
                .body("❌ Recurso no encontrado");
    }

    @ExceptionHandler(RuntimeException.class)
    public ResponseEntity<String> handleRuntime(RuntimeException ex) {
        logger.error("⚠️ Error inesperado en la aplicación", ex);
        return ResponseEntity.status(HttpStatus.INTERNAL_SERVER_ERROR)
                .body("⚠️ Error interno en el servidor: " + ex.getMessage());
    }
}
