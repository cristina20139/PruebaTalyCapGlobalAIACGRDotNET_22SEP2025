Pregunta 17 – Diferencia entre .java y .class

¿Qué diferencia hay entre un archivo .java y un archivo .class?

Respuesta:

- .java: Es el archivo de código fuente en Java. Contiene la implementación de clases, métodos y lógica del programa que escriben los desarrolladores. Este archivo es legible por humanos. - .class: Es el archivo compilado generado por el compilador de Java (javac). Contiene bytecode que puede ser ejecutado por la Java Virtual Machine (JVM), y no es directamente legible por humanos.

Ejemplo práctico:

1. Archivo MyClase.java

public class MyClase {
    public static void main(String[] args) {
        System.out.println("Hola mundo");
    }
}

2. Archivo MyClase.class

// Este archivo es generado por el compilador javac
// Contiene bytecode que se ejecuta en la JVM
// No es legible directamente como código Java
// Se ejecuta con: java MyClase
Nota: El flujo típico es escribir el código en .java, compilarlo con javac para obtener .class y luego ejecutarlo con la JVM.
