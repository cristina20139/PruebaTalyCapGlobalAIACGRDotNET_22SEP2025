Pregunta 11 – Opción Múltiple

En Java, ¿Qué diferencia hay entre una interfaz y una clase abstracta?

  • a) La interfaz se implementa en una clase, mientras que la clase abstracta se hereda en una clase
  • b) La clase abstracta requiere por lo menos un método abstracto, mientras que la interfaz no requiere métodos abstractos
  • c) Ambos conceptos son recíprocos
  • d) Todas las respuestas son correctas

Respuesta Correcta:

a) La interfaz se implementa en una clase, mientras que la clase abstracta se hereda en una clase

Justificación:

En Java, una interfaz define un contrato que una clase debe implementar mediante la palabra clave implements. Una clase abstracta puede contener métodos abstractos y métodos con implementación, y se hereda usando extends. La opción b) es incorrecta porque una clase abstracta no requiere necesariamente un método abstracto; puede tener solo métodos concretos. La opción c) es incorrecta ya que interfaz y clase abstracta no son recíprocos, sino conceptos distintos. Por eso, la opción correcta es a).

Nota: Las interfaces permiten múltiples implementaciones, mientras que una clase solo puede extender una clase abstracta.

Ejemplo en Java:

// Interfaz
interface Conducible {
    void conducir();
}

// Clase abstracta
abstract class Vehiculo {
    protected String marca;

    public Vehiculo(String marca) {
        this.marca = marca;
    }

    // Método concreto
    public void mostrarMarca() {
        System.out.println("Marca: " + marca);
    }

    // Método abstracto
    public abstract void tipoVehiculo();
}

// Clase que implementa la interfaz y hereda la clase abstracta
class Auto extends Vehiculo implements Conducible {
    public Auto(String marca) {
        super(marca);
    }

    @Override
    public void tipoVehiculo() {
        System.out.println("Tipo: Auto");
    }

    @Override
    public void conducir() {
        System.out.println("Conduciendo el auto...");
    }
}

public class Main {
    public static void main(String[] args) {
        Auto miAuto = new Auto("Toyota");
        miAuto.mostrarMarca();
        miAuto.tipoVehiculo();
        miAuto.conducir();
    }
}
